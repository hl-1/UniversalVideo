# Changelog

## v1.0.0 - 2026-07-05

- 发布 VideoDream 1.0.0 首个稳定版本。
- 支持 FastAPI 后端封装 yt-dlp 与浏览器嗅探解析。
- 支持 Vue 3 下载工作台、视频信息解析、格式选择、任务进度与本地另存为。
- 支持抖音公开视频浏览器解析、真实标题/作者展示与本地截图封面兜底。
- 支持 YouTube 浏览器解析与音视频合并兜底。
- 支持 PornHub 播放器源解析，通过页面上下文获取 signed MP4，避免 HLS 卡 95% 与无效 MP4 文件。
- 加入下载结果校验，避免图片、空响应或无效小文件被保存为 MP4。
